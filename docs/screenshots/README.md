# Screenshots

Referenced by the main `README.md`. Expected files:

- `shortcuts.png` — Shortcuts tab
- `keys.png` — Keys tab
- `apps.png` — Apps tab
- `trends.png` — Trends tab
- `menubar.png` — menu bar icon + popover (optional)
- `permission.png` — permission prompt state (optional)

## Capture

With the app running and the popover open, run from the repo root:

```bash
# Click the menu bar icon first, then switch to the tab you want, then run:
screencapture -iW docs/screenshots/shortcuts.png
# The cursor becomes a camera; click the popover window.
```

Repeat for each tab.
